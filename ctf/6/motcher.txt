numA dd 4.0
numB dd 8.0
numC dd 3.14
result dd 0.0

1) result = 2 * numC * numB + numA         b
2) result = numA + numC * numB * numB      a
3) result = (numB + numA) / numC           c
4) result = sin(numB + numA) * 2 * numC
5) result = sin(numC) * (numB + numA)      d

1-b, 2-a, 3-c, 4-e, 5-d

a) fld offset numC
   fld offset numB
   fld offset numB
   fmulp
   fmulp
   fadd offset numA
   fstp result
   
b) fld offset numC
   fld offset numB
   fld offset numB
   faddp
   fmulp
   fadd offset numA
   fstp result
   
c) fld offset numC
   fld offset numB
   fadd offset numA ; c (a+b)
   fxch             ; (a+b) c
   fdivp            ; (a+b) / c
   fstp result
   
d) fld offset numC
   fld offset numB
   fadd offset numA
   fxch
   fsin
   fmulp
   fstp result
   
e) fld offset numC
   fld offset numB
   fadd offset numA
   fsin
   fxch
   fadd offset numC
   fmulp
   fstp result
